# Think in java

* 对象 Object-Oriented programming
  
* 万物皆对象
   * A 程序是对象的集合，它们通过发送消息来告知彼此所要做
   * B 每个对象都有自己的由其他对象所构成的存储
   * C 每个对象都拥有类型
   * D 某一特定类型的所有对象都可以接受同样的消息(可替代性)

* 对象具有状态（数据成员）、行为（方法）和标识（内存地址）
  类描述了具有特性（数据成员）和行为（功能）的功能集合
